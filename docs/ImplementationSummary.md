## Technology Stack

### Backend
- Node.js v18+
- Express.js ^4.19.2
- MongoDB & Mongoose ^8.4.0
- JWT for authentication
- Bcrypt for password hashing

### Frontend
- React 18.3.1
- Vite 5.2.12
- React Router v6
- Axios for HTTP

### Testing
- Jest ^29.7.0 (Unit tests)
- Vitest ^0.34.0 (Component tests)
- Supertest ^6.3.3 (API tests)
- Cucumber.js for BDD

### Development Tools
- Nodemon for auto-reload
- ESLint for code quality
- Prettier for formatting
