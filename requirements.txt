## Installation Steps

### 1. Clone Repository
```bash
git clone https://github.com/A7MED505/SaaSPrototype.git
cd SaaSPrototype
```

### 2. Backend Setup
```bash
cd source_code
npm install
```

### 3. Frontend Setup
```bash
cd frontend
npm install
```

### 4. Environment Configuration
```bash
# In root directory
cp .env.example .env
# Edit .env with your configuration
```

### 5. Database Setup
- Local MongoDB:
  ```bash
  mongod  # Start MongoDB server
  ```
- Or use MongoDB Atlas cloud

## Verification

After installation, verify everything works:

```bash
# Terminal 1 - Backend
cd source_code
npm run dev
# Should show: Server running on http://localhost:8000

# Terminal 2 - Frontend
cd source_code/frontend
npm run dev
# Should show: ➜  Local:   http://localhost:5173/

# Terminal 3 - Tests
cd source_code
npm test  # Should pass all backend tests

cd source_code/frontend
npm test -- --run  # Should pass all frontend tests
```
